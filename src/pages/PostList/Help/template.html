Example Post:

Most parts of a post can also be hovered over for an explanation.

Opening Time

This is the time (in your selected timezone) that the match opens/whitelist is turned off.

The Start Time warning will show up if the time is a non-standard starting time (whitelist on).

Underneath the opening time is a relative time showing how long til the game opens (auto-updates)

Post Title

Example:

The title of the game, clicking this will take you to it's 'post page' where you can view the entire match post and find the server IP address

Team Style

Example:

This is the team style of the game. e.g. FFA, Chosen ToX, Random ToX

ToX means 'Teams of'. To4 = Teams of 4

Gamemodes/Scanrios

Example:

These are the scenarios that are in the game, checking out the post itself will give you a description/instructions for the scenario

Host + Region

Example: +

Which continent the server is hosted in as well as the Reddit name of the host

Clicking on the Reddit logo will add the host to your favourites list

Favourite/Blocked

Favourite hosts will show a Favourite or Blocked label in the top left to signify their status. The post will also be tinted green/red. Blocked hosts do not show up by default and need to be enabled in the post filters
