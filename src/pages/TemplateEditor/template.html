Help - Available Template Variables

Available Variables:

opensUTC

When the match opens in the format MMM DD HH:mm Z

title

The title of the post

region

The short code of the region

teams

The generated team type. e.g. Mystery To4

scenarios

Comma separated list of scenarios

How to use:

Surround the variable with curly brackets

**Match opens at** {{ opensUTC }} => Match opens at Apr 12 10:20 UTC
